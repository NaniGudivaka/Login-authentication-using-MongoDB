# User Authentication Frontend

A simple frontend for a user authentication system built with HTML, CSS, and JavaScript.

## Features

- User Signup
- User Login
- Dashboard
- Logout
- Form Validation
- API Integration

## Tech Stack

- HTML
- CSS
- JavaScript

## Run Locally

1. Clone the repository.
2. Open the project in VS Code.
3. Start Live Server or open `index.html`.

## Backend

This project connects to the authentication backend API.
